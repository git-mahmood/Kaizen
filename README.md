# KAIZEN — Discipline in Form

Premium streetwear rooted in the Shinobi path. The debut "Shinobi" collection.

## Stack

- Next.js 14 (App Router, Static Export)
- TypeScript
- Tailwind CSS
- Framer Motion
- Zustand

## Deploy

```bash
npm install
npm run build
```

Static output is generated in `./out` — deploy anywhere.

This repo includes a GitHub Actions workflow (`.github/workflows/deploy.yml`) that builds and deploys to GitHub Pages automatically on push to `main`.

**Required:** In your repo settings, set Pages source to **"GitHub Actions"**.

Settings → Pages → Source → GitHub Actions
